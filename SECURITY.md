# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

## Reporting a Vulnerability

Please report security vulnerabilities to **founder@trust-os.io**.

Do **not** open a public GitHub issue for security matters.

Include as much detail as possible:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and work with you on a coordinated disclosure.
